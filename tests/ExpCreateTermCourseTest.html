# ExpCreateTermCourseTest test

# Test Case

Creating an experiential term course.

# Requirements

  • An experiential education module.
  • The portal staff account needs to be an experiential admin.
  • At least one existing term.

# Test Data

# Attributes

# name object

The multilingual course name. See Translations.

# code object

The multilingual course code. See Translations.

# section object

The multilingual course section. See Translations.

# academic boolean

Whether the course is an academic course or not.

# term text

The term to select from the dropdown.

# description object

The multilingual term course description. See Translations.

# website text

The course website.

# contactEmail text

The course contact email.

# manager object

The manager to lookup and select. See Lookup Search.

# manager object

The faculty member to lookup and select. See Lookup Search.

# expType object

The ID of the experience type we want to add to the term course.
