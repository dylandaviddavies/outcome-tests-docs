# Large Text Answer <Badge text="object" vertical="middle" />
The answer for a [LargeText](./acrm-question-type/#large-text) type User Registration Form Question.

## Attributes
### value <Badge text="string" vertical="middle" />
The text value.

## Usage
``` json
{
    "question": {
        "id": "101",
        "type": "LargeText"
    },
    "value": "Hello, World"  
}
```
