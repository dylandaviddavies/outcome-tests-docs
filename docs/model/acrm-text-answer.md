# Text Answer <Badge text="object" vertical="middle" />
The answer for a [Text](./acrm-question-type/#text) type User Registration Form Question.

## Attributes
### value <Badge text="string" vertical="middle" />
The text value.

## Usage
``` json
{
    "question": {
        "id": "101",
        "type": "Text"

    "value": "Hello, World"  
}
```
